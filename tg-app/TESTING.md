# TESTING.md — Руководство тестировщика

**Проект:** Telegram Mini App «Ana Krista · Таро»  
**Стек:** HTML + CSS + JS (без фреймворков) + Vercel Serverless  
**Тестировать в:** браузере (Chrome/Safari) И в Telegram (обязательно оба)

---

## 1. Быстрый старт

### Живые URL для тестирования

| Среда | URL |
|---|---|
| Веб-версия | `https://tg-app-nine-lovat.vercel.app` |
| Telegram Mini App | `t.me/AnaKristaTaro_Bot/taro` |
| Проверка webhook | `https://tg-app-nine-lovat.vercel.app/api/webhook` (GET) |
| Бот напрямую | `t.me/AnaKristaTaro_Bot` |

### Ожидаемый ответ на `GET /api/webhook`
```json
{ "ok": true, "status": "Bot webhook is running ✓" }
```
Если ответ другой — webhook упал, тест прекращать нет смысла.

## 2. Окружение

### Запуск локально

```bash
git clone https://github.com/ejsmile2013-2026/tarot-digital-products.git
cd tarot-digital-products/tg-app
# Никаких npm install не нужно — чистый HTML/CSS/JS
# Открой index.html через любой локальный сервер:
npx serve .
# или
python3 -m http.server 3000
# Открыть: http://localhost:3000
```

### Переменные окружения (для webhook)

Скопируй `.env.example` в `.env` и заполни:
```
TELEGRAM_BOT_TOKEN=<токен от @BotFather>
ADMIN_CHAT_ID=449959218
```
Без этих переменных webhook работает, но уведомления не отправляет.

### LocalStorage — ключи которые влияют на поведение

| Ключ | Когда устанавливается | Что делает |
|---|---|---|
| `ana_onboard_shown` | После нажатия «Начать →» в онбординге | Скрывает онбординг при следующих открытиях |
| `ana_offer_shown` | После нажатия «Получить скидку 15%» или «Пропустить» | Скрывает оффер при следующих открытиях |

**Сброс для повторного тестирования модалок:**
```
DevTools → Application → Local Storage → удалить ключи
```

## 3. Структура приложения

### Экраны

| ID | Название | Как открыть |
|---|---|---|
| `home` | Главная / Каталог | Стартовый экран |

## 8. Известные ограничения (не баги)

| Ограничение | Причина |
|---|---|
| Оплата не реализована | Telegram Payments не подключены (v2) |
| Файлы продуктов не отправляются автоматически | Нет хранилища файлов (v2) |
| История заказов отсутствует | Нет базы данных (v2) |
| Категория «Курсы» в нижнем меню отсутствует | Нет услуг с `category: 'courses'` в `data.js` |
| Telegram BackButton не работает в браузере | SDK-ограничение, заменён HTML-кнопкой «‹ Назад» |
| `sendData()` не вызывается | Заменён на `fetch('/api/webhook')` чтобы Mini App не закрывался |
